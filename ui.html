Advanced Options:

instructions

  • Design your page numbers however you want, using placeholder numbers.
  • Make sure every layer containing a page number is named "page number". If you've already used a different layer name for all your page numbers, you can change it in the Advanced Options.
  • Every page has to be framed (although you can turn off the "Ignore non-frame nodes" option in the Advanced Options).
  • Arrange the pages in rows, left to right.
  • Select all the slides or pages. Select even the slides that don't have a page number, like full-bleed image slides, so they are still counted
  • Run the plugin.

For more detailed instructions and the source code, visit our GitHub README.
